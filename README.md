# CodeAlpha Language Translation Tool

A simple and user-friendly language translation web application developed using Python and Flask. This project allows users to translate text from one language to another instantly.

## Features
- Translate text between multiple languages
- Clean and responsive user interface
- Fast and accurate translations
- Built with Python and Flask

## Technologies Used
- Python
- Flask
- HTML
- CSS
- JavaScript
- Google Translate API / deep-translator

## How to Run
1. Install Python
2. Install the required packages:
   pip install flask deep-translator
3. Run the application:
   python app.py
4. Open http://127.0.0.1:5000/ in your browser
